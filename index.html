About the Plugin

Obsidian Google Calendar Sync is a plugin for Obsidian that syncs your Obsidian Tasks to Google Calendar as events. It supports syncing reminders, task start/end times, and has auto-sync functionality.

The plugin uses IDs in the form of HTML comments included as part of the task content to reliably track tasks across different lines, files, app reboots or even across vaults.

Installation

The plugin can be installed manually from the GitHub Releases page.

  1. Download the latest release
  2. Extract the zip file into your Obsidian vault's .obsidian/plugins/ directory
  3. Restart Obsidian and enable the plugin in Settings > Community Plugins
